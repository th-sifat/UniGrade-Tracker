About UniGrade Tracker

Our Mission

This project aims to simplify CGPA tracking for students and provide an easy interface to record and review your grades and cumulative history.

Key Features

  • Accurate SGPA Calculation
  • Detailed Mark Breakdown
  • Securely Save Your History
  • Track Cumulative CGPA

About the Project

This UniGrade Tracker was built as a Web Programming course project to help students calculate and keep track of semester and cumulative GPAs.
